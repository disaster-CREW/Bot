Privacy Policy

1. Overview

This Privacy Policy explains what information the bot processes and how it is used. By using the bot, you agree to the practices described here.

2. Information the Bot Processes

The bot only processes the minimum information required to function, including:

  • Command inputs (e.g., slash commands)
  • User IDs (only temporarily while responding)
  • Server IDs (for command context)

No personal data is stored or logged.

3. Data Storage

The bot does not store:

  • Messages
  • Usernames or nicknames
  • DM content
  • IP addresses
  • Any personal identifying information

All processing is temporary and only happens in real time.
